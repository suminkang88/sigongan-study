## 1. Data
### 데이터의 정의
- 현실에서 관찰하거나 측정한 사실을 기호, 숫자, 문자로 기록한 것
- 추측이나 예측을 위한 기반으로 활용되는, 알려진 사실 또는 알려진 것
- 데이터 자체는 의미가 없으며, 맥락과 해석이 붙을 때 의미가 생김
	- 데이터 - (맥락) - 정보 - (패턴) - 지식
		- 예: 23°C (데이터) → 오늘 서울 기온 (정보) → 이번 달 평균 기온 (지식)

### 데이터의 종류 
**저장 형태에 따라** 
- **정형 데이터**: 행과 열로 정리된 구조화된 데이터. 엑셀, DB 테이블. 빠른 분석 가능
- **반정형 데이터**: 일정한 구조가 있지만 완전하지 않음. JSON, XML 등
-  **비정형 데이터**: 정해진 형식 없음. 텍스트, 이미지, 영상, SNS 게시물 등 

**성격에 따라** 
- **정량적 데이터**: 숫자로 측정 가능한 값. 판매량, 온도, 수익
- **정성적 데이터**: 수치화하기 어려운 값. 고객 리뷰, 감정, 의견

### 어디에, 어떻게 존재하는가?
- 모든 데이터는 결국 비트(0과 1)로 표현됨
	- 0과 1 - 비트 - 바이트 - 텍스트/이미지/영상
- **HDD (Hard Disk Drive)**: 자석 원리로 데이터를 저장. 물리적으로 회전하는 디스크에 자기장으로 0과 1을 기록. 용량 대비 저렴하지만 충격에 약하고 속도가 느림
- **SSD (Solid State Drive)**: 움직이는 부품 없이 전기 신호로 데이터를 저장. HDD보다 빠르고 가볍지만 가격이 비쌈. 현재 대부분의 노트북/PC에 사용
- **클라우드**: 인터넷을 통해 외부 서버에 데이터를 저장하는 방식. 내 컴퓨터에 저장하는 게 아니라 AWS, Google, Microsoft 같은 기업의 데이터센터 서버에 저장됨. 어디서든 접근 가능하고 용량 제한이 유연하지만 인터넷 연결이 필요함

셋 모두 데이터를 저장하는 수단이지만, 위치와 접근 방식이 다르다.
HDD와 SSD는 **내 컴퓨터 안에 물리적으로 존재**하는 저장장치이고, 클라우드는 **인터넷 너머 남의 서버**에 저장하는 방식이다. 즉 HDD/SSD는 로컬 저장, 클라우드는 원격 저장이다.

클라우드 서버도 결국 어딘가의 데이터센터에 있는 HDD나 SSD에 데이터를 저장한다. 클라우드가 별도의 저장 기술인 게 아니라, HDD/SSD로 이루어진 거대한 저장 공간을 인터넷으로 빌려 쓰는 개념이다.

## 2. Data Analyst (데이터 분석가) 
### 데이터 분석가의 정의
데이터에서 인사이트를 발견해 조직의 의사결정을 돕는 사람 

### 하는 일 
- 데이터 수집 및 전처리 (불필요한 데이터 제거, 정제)
- 데이터 분석 및 패턴 탐색 
- 시각화: 보고서, 대시보드 제작
- 분석 결과를 경영진 / 타 부서에 전달 
- 문제 정의 → 가설 설정 → 데이터 분석의 흐름으로 일함

### 기업에 왜 필요한가? 
**감**이 아닌 **데이터 기반의 근거**로 의사결정을 내리기 위해서. 
- 어떤 제품이 잘 팔리는지, 왜 고객이 이탈하는지 수치로 파악 
- 시장 점유율 예측, 가격 책정, 운송비 최적화 등에 활용
- 잘못된 직관에 의존하는 의사결정 리스크를 줄임

### 필요한 역량 
- SQL: 데이터베이스에서 데이터 추출 
- Python / R: 데이터 분석 및 시각화 
- 통계 지식: 패턴과 관계를 수치로 해석 
- 커뮤니케이션: 분석 결과를 비개발자에게도 설명할 수 있어야 함 
- 문제 정의 능력: 무엇을 분석할지 먼저 명확히 알아야 함
